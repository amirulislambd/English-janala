# English Janala - Learn English Vocabulary Easily

![English Janala Hero Section](https://i.ibb.co.com/7NdG6HVm/English-Janala.png) 

**English is Easy!!** — A simple and interactive English vocabulary learning web app for beginners.

## 📋 Project Overview

**English Janala** is a beginner-level frontend web application designed to help Bangladeshi students learn English vocabulary in an easy and fun way.  

After logging in, users can browse vocabulary words with Bengali meanings, listen to correct pronunciation, and view detailed information.

This project was built as my early learning project to practice JavaScript DOM manipulation, event handling, and responsive UI.

## ✨ Key Features

- **Login System**  
  Simple login form (username/password validation using JavaScript)

- **Vocabulary Learning Dashboard**  
  - Grid of vocabulary cards (Lesson-wise)
  - Each card shows:  
    - English Word  
    - Bengali Meaning  
    - Pronunciation guide

- **Interactive Icons**  
  - **Speaker Icon** → Click to hear the word’s pronunciation (audio playback)
  - **Details Icon** → Click to open a modal with more information (example sentences, usage, etc.)

- **Search Functionality**  
  Real-time search bar to filter vocabulary words

- **Lesson Selection**  
  Multiple lessons (Lesson-1 to Lesson-7) with categorized words

- **Fully Responsive Design**  
  Works perfectly on mobile, tablet, and desktop

- **Clean & Beginner-Friendly UI**  
  Built with Tailwind CSS for modern look
